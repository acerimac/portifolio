My Skills

Here are a few technologies I've been working with recently:

HTML

HTML (Linguagem de Marcação de Hipertexto) é o esqueleto fundamental da web, usado para estruturar e organizar o conteúdo de páginas, definindo o que é um título, parágrafo, imagem, link ou botão, servindo como base para a exibição de qualquer site.

JavaScript

JavaScript (JS) é uma linguagem de programação de alto nível amplamente utilizada para criar interatividade e comportamento dinâmico em sites.

CSS

O CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é a linguagem utilizada para definir a aparência e o design de uma página da web.

Python

Python é uma linguagem de programação de alto nível, interpretada e de propósito geral, conhecida por sua sintaxe clara e legibilidade, amplamente utilizada em desenvolvimento web, ciência de dados, automação e inteligência artificial.

SQL

SQL (Structured Query Language) é uma linguagem de programação padrão usada para gerenciar e manipular bancos de dados relacionais, permitindo a criação, consulta, atualização e exclusão de dados de forma eficiente.

Power BI

Power BI é uma ferramenta de análise e visualização de dados da Microsoft, utilizada para criar dashboards e relatórios interativos a partir de dados estruturados e não estruturados.
