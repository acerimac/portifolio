Here are a few technologies I've been working with recently:

HTML

HTML (Linguagem de Marcação de Hipertexto) é o esqueleto fundamental da web, usado para estruturar e organizar o conteúdo de páginas, definindo o que é um título, parágrafo, imagem, link ou botão, servindo como base para a exibição de qualquer site.

JavaScript

JavaScript (JS) é uma linguagem de programação de alto nível amplamente utilizada para criar interatividade e comportamento dinâmico em sites.

CSS

O CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é a linguagem utilizada para definir a aparência e o design de uma página da web.
